# MemeForge 🎭

> Free AI-Powered Meme Generator - Create viral memes in seconds!

![MemeForge](https://img.shields.io/badge/MemeForge-v1.0-blue) ![License](https://img.shields.io/badge/License-MIT-green) ![Stars](https://img.shields.io/github/stars/wcq1976/meme-forge)

## ✨ Features

- 🎨 **30+ Trending Templates** - Drake, Distracted Boyfriend, SpongeBob, and more
- 🤖 **AI-Powered Captions** - Get smart meme caption suggestions instantly
- 📷 **Photo Upload** - Turn your photos into custom memes
- ✍️ **Text-to-Meme** - Add custom top/bottom text to any template
- ⬇️ **HD Download** - Export as PNG or JPG in high quality
- 📤 **Easy Share** - Share directly to social media
- 🌐 **100% Free** - No signup, no credits, no limits

## 🚀 Quick Start

Visit **[https://memeforge.pages.dev](https://memeforge.pages.dev)** and start creating!

Or run locally:

```bash
# Clone the repo
git clone https://github.com/wcq1976/meme-forge.git
cd meme-forge

# Open in browser
open index.html
```

## 🛠️ Tech Stack

- **Frontend**: HTML5, Tailwind CSS, Vanilla JavaScript
- **AI**: MiniMax API (for caption suggestions)
- **Hosting**: Cloudflare Pages, GitHub Pages
